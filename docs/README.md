# BhuChain: Blockchain-Based Land Registration System

BhuChain is a decentralized application (DApp) designed to modernize land registration in Nepal. By leveraging Ethereum blockchain technology and a hybrid data management strategy, the system provides a tamper-proof "Source of Truth" for land ownership while ensuring high performance and user privacy.
